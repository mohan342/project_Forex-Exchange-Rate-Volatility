# Forex Exchange Rate Volatility

This project analyzes and predicts forex exchange rate volatility using AI and machine learning techniques.  
It includes data collection, preprocessing, exploratory data analysis, feature engineering, model training, evaluation, and visualization.

## Project Objectives
- Analyze exchange rate volatility patterns.
- Identify important market features affecting volatility.
- Compare machine learning models for prediction.
- Visualize key trends using charts and heatmaps.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost
- TensorFlow / Keras

## Project Workflow
1. Data collection
2. Data cleaning and preprocessing
3. Exploratory data analysis
4. Feature engineering
5. Model training
6. Prediction and evaluation
7. Visualization and interpretation

## Files Included
- `notebook.ipynb` - Main Colab notebook
- `model_comparison.csv` - Model results
- `forex_data_processed.csv` - Processed dataset
- `presentation.pptx` - Final presentation

## Results
The project compares multiple models and identifies the best-performing approach for forecasting forex volatility.
